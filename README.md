# Project-E Fitness Gym | Web Management System

The official Web Management System for Project-E Fitness Gym. This Next.js application serves as the central administrative hub for Gym Managers and Admins to handle daily operations, manage memberships, track attendance analytics, and oversee staff.

This website is one half of the Project-E hybrid ecosystem, working in tandem with the offline-first Flutter tablet app used at the front desk.

## 🏗 Architecture & Tech Stack

This project is built with a modern, type-safe stack designed for speed and maintainability:

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Backend:** Supabase (PostgreSQL, Row Level Security, RPC functions)
- **Authentication:** `@supabase/ssr` (Server-Side Rendering cookie-based auth)

## 🚀 Getting Started

Follow these instructions to set up the project on your local development machine.

### Prerequisites

- **Node.js** (v18.x or higher recommended)
- **npm** (or yarn/pnpm)

### 1. Clone & Install dependencies

```bash
# Clone the repository
git clone [your-repo-url]
cd project-e-management-system

# Install dependencies
npm install
```
